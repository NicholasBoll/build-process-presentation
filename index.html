    • When should I start?

    My favorite?

  • Gulp!

    • Example pipeline

    • Transpile CoffeeScript to JavaScript (gulp-coffee)
    • Transpile Stylus to CSS (gulp-stylus)
    • Annotate AngularJS (gulp-ng-annotate)
    • Template cache (gulp-angular-templatecache)
    • Concat resources (JS, CSS) (gulp-concat)
    • Minify resources (JS, CSS, SVG, HTML, IMG, etc) (gulp-uglify)
    • GZip (gulp-gzip faster than runtime compression)
    • Inject scripts, styles, etc (gulp-inject)

    Sourcemaps

    • gulp-sourcemaps
    • Maps transforms from each stream transform!

    Optimizing for dev

    • Process only changed files
    • gulp-changed
    • gulp-newer
    • gulp-cached
    • gulp-remember
